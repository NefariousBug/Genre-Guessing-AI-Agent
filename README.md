Prvo treba kreirati venv okruzenje (python -m venv venv), aktivirati ga (venv\Scripts\Activate), te unutar njega instalirati iz requirements.txt (pip install -r requirements.txt). Osim requirements.txt potreban je ffmpeg koji se instalira na nivou sistema a ne unutar venv-a pa ga nisam ukljucio u requirements.txt.
Komanda unutar powershella: winget install ffmpeg  ili ako imate chocolatey: choco install ffmpeg
Ako ne postoji genre_model.pkl, pokrenuti train_model.py skriptu. Ona kreira pocetnog pre-trained agenta. Posto je dataset preko 1gb velik ostavio sam samo features csv fajl preko kojeg se trenira pocetni model, on je rezultat obrade dataseta. Postoji file extract_features.py koji vrsi obradu dataseta ali sam ga iskljucio da ne bi doslo do problema zato sto ne mogu uploadati dataset jer je pre velik, tako da kao sto sam rekao imamo samo rezultat te obrade - gtzan_features.csv. Ali u biti logika ekstrakcije je identicna u services.py za dodatne pjesme koje korisnik testira.
Za pokretanje agenta pokrenuti skriptu app.py koja ce startati flask aplikaciju i izbaciti link za web browser koji vodi do front enda.
